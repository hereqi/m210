Cloud

Was ist Cloud?

Die Cloud stellt Ressourcen wie Rechenleistung, Speicher und Anwendungen über das Internet bereit.
Anwender können auf diese Ressourcen zugreifen, ohne etwas lokal auf ihren Geräten zu installieren.

Vorteile der Cloud-Nutzung

  1. Skalierbarkeit: Ressourcen können einfach angepasst werden.
  2. Kosteneffizienz: Nutzer zahlen nur für die tatsächlich genutzten Ressourcen.
  3. Zugänglichkeit: Von überall und von verschiedenen Geräten aus zugänglich.
  4. Flexibilität: Unterstützung für verschiedene Anwendungen und Workloads.

Beispiele für Cloud-Dienste

  • Amazon Web Services (AWS)
  • Google Cloud Platform (GCP)
  • Microsoft Azure
